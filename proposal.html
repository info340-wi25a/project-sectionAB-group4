Problem Description

Overview

Many people own tools or equipment (e.g., ladders, power washers, party supplies) that they use infrequently. Meanwhile, others need these tools for short-term use but are unwilling to buy them due to high costs or limited storage. Existing rental platforms (like Home Depot rentals) often have limited inventory and higher prices. Additionally, local peer-to-peer sharing platforms are fragmented or nonexistent.

Information Problem

Thereâ€™s no cohesive solution enabling peer-to-peer renting for underutilized tools and equipment in local communities. Users face challenges finding nearby tools at reasonable prices or renting their own unused items securely. Current solutions lack seamless user interfaces, proper data organization, and persistence.

Existing Solutions

  • Home Depot Tool Rentals: Expensive for short-term use; lacks user-to-user interaction.
  • Facebook Marketplace/Nextdoor: Offers some rentals but lacks structured booking and secure processes.

App Description

Proposed Solution

Rent-A-Tool is a web application connecting users looking to rent tools or equipment with those offering unused items. The platform provides an intuitive interface for browsing, booking, and managing rentals with secure and persistent data storage.

Key Features

  1. Interactive Catalog of Tools (Feature 1)
    • Users can browse tools available in their area using filters (e.g., category, price, availability).
  2. Rental Booking System (Feature 2)
    • Users can request bookings for tools, specifying rental dates and times.
    • Tool owners receive notifications and can accept/decline requests.
  3. User Reviews and Ratings (Feature 3 - Half)
    • After completing a rental, users can rate their experience and provide feedback.
    • Ratings help maintain quality and trust within the community.

Functionality Walkthrough

1. Homepage & Tool Catalog

User Action: Users land on a homepage featuring an introduction to the platform and a tool catalog. They can browse tools or log in for more features.

2. Tool Details Page

User Action: Clicking on a tool displays its details, including descriptions, photos pricing, availability, and user review. A "Book Now" button initiates the booking process.

3. User Authentication

User Action: Users must log in or create an account to rent or list tools.

4. Booking Tools

User Action: Logged-in users can select rental dates from an interactive calendar and click "Book Now." The tool owner received a notification of the booking request and can approve or deny it.
